# flappy-bird-oyunu
Borulara çarparsak oyun yeniden başlar. Amacımız borulardan çarpmadan başarıyla geçerek puan kazanmaktır.

# Kullanılan Teknolojiler
Html , CSS , JavaScript

# Nasıl Oynanır
• Enter'a basmanız gereken ekranda basıp çalıştırıp daha sonrasında kuşun yukarı doğru ivme alması için **yukarı yön tuşu (↑)** kullanmanız gerekir.
• Borulara çarpmadan arasından geçerek skor toplamanız gerekmektedir.

# Nasıl Çalışır
Sunucuya gerek kalmadan `index.html` dosyasını herhangi bir tarayıcıdan çalıştırmanız yeterli olacaktır.
